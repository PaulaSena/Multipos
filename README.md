# Valores múltiplos de 3

```
        #region  Valores multiplos de 3
				int n;
        Console.WriteLine("Digite um numero");
        n = Convert.ToInt32(Console.ReadLine());
        for (int i = 1; i <= n; i++)
        {
            if (i % 3 == 0)
            {
                Console.Write("{0} | ", i);

            }

        }
        Console.ReadKey();

        #endregion
```
